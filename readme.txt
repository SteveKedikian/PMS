# PMS
Project Management System


File formats for components:

PROJECT FORMAT -       title /-/ date

TASK FORMAT -         title /-/
                      description /-/

USER FORMAT -         name /-/ age




Actions User can do:

UPDATE
update projects set 'member'="name" where 'member'="name"

example: update projects set title="Kilimanjaro" where date="13/11/5211"
